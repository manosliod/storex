# Yarn Installation

```bash
npm install --global yarn 
````

## Project requirements

Use the package manager **yarn** to install the required libraries.

```bash
yarn install
```

## Usage

For DEV
```bash
yarn dev
```

For PROD
```bash
yarn build
yarn start
```
